# 项目结构与文件职责说明

## 1. 根目录

| 路径 | 职责 |
|---|---|
| `apps/` | 前端和后端应用源码。 |
| `blockchain/` | 智能合约、Hardhat 配置和部署脚本。 |
| `docs/` | 项目说明文档。 |
| `scripts/` | 启动、部署、重置和辅助脚本。 |
| `verifier/` | 独立验真工具。 |
| `package.json` | 根命令入口。 |
| `README.md` | 项目总览入口。 |

## 2. 后端 `apps/backend`

| 路径 | 职责 |
|---|---|
| `apps/backend/src/index.js` | 业务后端入口，注册中间件、路由和后台任务。 |
| `apps/backend/src/auth-server.js` | 独立认证服务入口。 |
| `apps/backend/src/db.js` | 业务数据库建表、连接和持久化。 |
| `apps/backend/src/user-db.js` | 用户数据库建表、连接和持久化。 |
| `apps/backend/src/auth.js` | JWT 鉴权和角色校验。 |
| `apps/backend/src/logger.js` | 结构化日志写入。 |
| `apps/backend/src/notifications.js` | 站内通知写入和查询辅助。 |
| `apps/backend/src/onchain-operations.js` | 链上操作台账管理。 |
| `apps/backend/src/permit.js` | 链上 permit 签名和参数哈希。 |
| `apps/backend/src/ipfs.js` | IPFS 上传和 Gateway 地址处理。 |
| `apps/backend/src/listing-public-state.js` | 房源公开状态推导。 |
| `apps/backend/src/app-error.js` | 统一错误响应结构。 |

### 2.1 后端路由

| 路径 | 职责 |
|---|---|
| `apps/backend/src/routes/auth.js` | 注册、登录、资料、头像、钱包绑定。 |
| `apps/backend/src/routes/listings.js` | 房源发布、列表、详情、编辑、上链、反馈。 |
| `apps/backend/src/routes/contracts.js` | 合同申请、签署、支付、续约、提前解约、PDF、评价。 |
| `apps/backend/src/routes/verify.js` | 平台房源验真和合同验真。 |
| `apps/backend/src/routes/notifications.js` | 通知列表、未读数和已读操作。 |

### 2.2 后端运行数据

| 路径 | 职责 |
|---|---|
| `apps/backend/data/database.sepolia.sqlite` | Sepolia 业务数据库。 |
| `apps/backend/data/database.local.sqlite` | Local 业务数据库。 |
| `apps/backend/data/users.sepolia.sqlite` | Sepolia 用户数据库。 |
| `apps/backend/data/users.local.sqlite` | Local 用户数据库。 |
| `apps/backend/data/uploads/` | 上传图片存储目录。 |

运行数据不提交到 Git。

## 3. 前端 `apps/frontend`

| 路径 | 职责 |
|---|---|
| `apps/frontend/src/main.jsx` | React 挂载入口。 |
| `apps/frontend/src/app/App.jsx` | 路由、布局、网络状态和通知入口。 |
| `apps/frontend/src/app/providers/AuthContext.jsx` | 登录态、用户信息、头像、网络偏好和钱包连接。 |
| `apps/frontend/src/shared/api/api.js` | API 请求封装。 |
| `apps/frontend/src/shared/blockchain/RentalChainABI.json` | 前端链交互 ABI。 |
| `apps/frontend/src/shared/listingUtils.js` | 房源展示工具函数。 |
| `apps/frontend/src/index.css` | 全局样式。 |

### 3.1 前端页面

| 页面 | 职责 |
|---|---|
| `HomePage.jsx` | 首页。 |
| `LoginPage.jsx` | 登录和注册。 |
| `ListingsPage.jsx` | 房源列表、智能搜索和区域筛选。 |
| `ListingDetail.jsx` | 房源详情、申请签约和房源反馈。 |
| `ListingReviewsPage.jsx` | 房源评论和租后评价列表。 |
| `PublishListing.jsx` | 发布房源。 |
| `MyListings.jsx` | 房东房源管理。 |
| `MyContracts.jsx` | 我的合同列表。 |
| `ContractPage.jsx` | 合同详情、签署、支付、续约、提前解约和评价。 |
| `VerifyPage.jsx` | 平台验真页面。 |
| `NotificationsPage.jsx` | 通知中心。 |
| `ProfilePage.jsx` | 个人资料和头像。 |

## 4. 区块链 `blockchain`

| 路径 | 职责 |
|---|---|
| `blockchain/contracts/RentalChain.sol` | 主业务合约。 |
| `blockchain/scripts/deploy_rental.js` | 合约部署脚本。 |
| `blockchain/hardhat.config.js` | Hardhat 网络和编译配置。 |
| `blockchain/deployments-rental-sepolia.json` | Sepolia 部署结果。 |
| `blockchain/deployments-rental-localhost.json` | Local 部署结果。 |
| `blockchain/.ganache-db/` | Local 持久化链数据目录。 |

部署结果和本地链数据按运行环境生成。

## 5. 独立验真工具 `verifier`

| 路径 | 职责 |
|---|---|
| `verifier/server.js` | 独立验真 Web 服务。 |
| `verifier/web/index.html` | 独立验真页面结构。 |
| `verifier/web/app.js` | 独立验真前端逻辑。 |
| `verifier/web/styles.css` | 独立验真样式。 |
| `verifier/scripts/verify-contract-pdf.js` | 合同 PDF 验真逻辑。 |
| `verifier/scripts/verify-listing.js` | 房源验真和房源详情逻辑。 |
| `verifier/lib/runtime.js` | 合约配置、运行配置和 RPC 解析。 |
| `verifier/config/runtime.json` | 独立验真运行配置。 |

## 6. 脚本 `scripts`

| 路径 | 职责 |
|---|---|
| `scripts/sync_abi_and_deployment.js` | 同步 ABI、部署地址和 verifier 配置。 |
| `scripts/test_env_isolation.js` | Sepolia / Local 环境隔离检查。 |
| `scripts/reset-user-risk.js` | 重置用户风控计数。 |
| `scripts/seed-local-listings.js` | 向 Local 数据库写入演示用户和北京房源数据。 |
| `scripts/ps1/` | Windows PowerShell 脚本。 |
| `scripts/mac/` | macOS shell 脚本。 |

### 6.1 macOS 脚本

| 路径 | 职责 |
|---|---|
| `scripts/mac/start-demo.sh` | 一键启动 macOS 演示环境：本地链、认证服务、双后端、前端和独立 verifier。 |
| `scripts/mac/start-sepolia-services.sh` | 启动认证服务、Sepolia 后端和前端。 |
| `scripts/mac/start-local-services.sh` | 启动认证服务、Local 后端和前端。 |
| `scripts/mac/start-parallel-services.sh` | 同时启动认证服务、Sepolia 后端、Local 后端和前端。 |
| `scripts/mac/start-persistent-local-node.sh` | 启动 Ganache 本地持久化链。 |
| `scripts/mac/reset-sepolia-and-redeploy.sh` | 重置 Sepolia 数据并重新部署合约。 |
| `scripts/mac/reset-local-and-redeploy.sh` | 重置 Local 数据、重建本地链并重新部署合约。 |
| `scripts/mac/setup-local-ipfs.sh` | 下载项目内 Kubo 二进制。 |
| `scripts/mac/start-local-ipfs.sh` | 启动本地 IPFS 节点。 |
| `scripts/mac/start-verifier.sh` | 启动独立验真工具 Web 页面。 |
