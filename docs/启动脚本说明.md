# 启动脚本说明

## 1. Windows PowerShell 脚本

| 脚本 | 用途 |
|---|---|
| `scripts/ps1/start-sepolia-services.ps1` | 启动认证服务、Sepolia 后端和前端。 |
| `scripts/ps1/start-local-services.ps1` | 启动认证服务、Local 后端和前端。 |
| `scripts/ps1/start-parallel-services.ps1` | 同时启动认证服务、Sepolia 后端、Local 后端和前端。 |
| `scripts/ps1/start-persistent-local-node.ps1` | 启动 Ganache 本地持久化链。 |
| `scripts/ps1/reset-sepolia-and-redeploy.ps1` | 重置 Sepolia 数据并重新部署合约。 |
| `scripts/ps1/reset-local-and-redeploy.ps1` | 重置 Local 数据、重建本地链并重新部署合约。 |
| `scripts/ps1/setup-local-ipfs.ps1` | 下载项目内 Kubo 二进制。 |
| `scripts/ps1/start-local-ipfs.ps1` | 启动本地 IPFS 节点。 |
| `scripts/ps1/start-verifier.ps1` | 启动独立验真工具 Web 页面。 |

### 1.1 Sepolia 单环境
```powershell
powershell -ExecutionPolicy Bypass -File scripts/ps1/start-sepolia-services.ps1
```

启动端口：

- 认证服务：`3005`
- Sepolia 后端：`3000`
- 前端：`3001`

### 1.2 Local 单环境
```powershell
powershell -ExecutionPolicy Bypass -File scripts/ps1/start-local-services.ps1
```

启动端口：

- 认证服务：`3005`
- Local 后端：`3002`
- 前端：`3001`

Local 服务需要先启动本地链：

```powershell
powershell -ExecutionPolicy Bypass -File scripts/ps1/start-persistent-local-node.ps1
```

### 1.3 双环境并行
```powershell
powershell -ExecutionPolicy Bypass -File scripts/ps1/start-parallel-services.ps1
```

启动端口：

- 认证服务：`3005`
- Sepolia 后端：`3000`
- Local 后端：`3002`
- 前端：`3001`

### 1.4 本地链节点
```powershell
powershell -ExecutionPolicy Bypass -File scripts/ps1/start-persistent-local-node.ps1
```

本地链参数：

- RPC：`http://127.0.0.1:8545`
- Chain ID：`31337`
- 数据目录：`blockchain/.ganache-db`
- 默认账户数：`10`
- 默认余额：`1000 ETH`

### 1.5 重置与重部署
```powershell
powershell -ExecutionPolicy Bypass -File scripts/ps1/reset-sepolia-and-redeploy.ps1
powershell -ExecutionPolicy Bypass -File scripts/ps1/reset-local-and-redeploy.ps1
```

### 1.6 IPFS
```powershell
powershell -ExecutionPolicy Bypass -File scripts/ps1/setup-local-ipfs.ps1
powershell -ExecutionPolicy Bypass -File scripts/ps1/start-local-ipfs.ps1
```

### 1.7 独立验真工具
```powershell
powershell -ExecutionPolicy Bypass -File scripts/ps1/start-verifier.ps1
```

访问地址：`http://127.0.0.1:3010`

## 2. macOS shell 脚本

| 脚本 | 用途 |
|---|---|
| `scripts/mac/start-sepolia-services.sh` | 启动认证服务、Sepolia 后端和前端。 |
| `scripts/mac/start-local-services.sh` | 启动认证服务、Local 后端和前端。 |
| `scripts/mac/start-parallel-services.sh` | 同时启动认证服务、Sepolia 后端、Local 后端和前端。 |
| `scripts/mac/start-persistent-local-node.sh` | 启动 Ganache 本地持久化链。 |
| `scripts/mac/reset-sepolia-and-redeploy.sh` | 重置 Sepolia 数据并重新部署合约。 |
| `scripts/mac/reset-local-and-redeploy.sh` | 重置 Local 数据、重建本地链并重新部署合约。 |
| `scripts/mac/setup-local-ipfs.sh` | 下载项目内 Kubo 二进制。 |
| `scripts/mac/start-local-ipfs.sh` | 启动本地 IPFS 节点。 |
| `scripts/mac/start-verifier.sh` | 启动独立验真工具 Web 页面。 |
| `scripts/mac/start-demo.sh` | 一键启动 macOS 演示环境，包括本地链、认证服务、双后端、前端和独立验真工具。 |

### 2.1 Sepolia 单环境
```bash
bash scripts/mac/start-sepolia-services.sh
```

### 2.2 Local 单环境
```bash
bash scripts/mac/start-local-services.sh
```

### 2.3 双环境并行
```bash
bash scripts/mac/start-parallel-services.sh
```

### 2.4 本地链节点
```bash
bash scripts/mac/start-persistent-local-node.sh
```

### 2.5 重置与重部署
```bash
bash scripts/mac/reset-sepolia-and-redeploy.sh
bash scripts/mac/reset-local-and-redeploy.sh
```

### 2.6 IPFS
```bash
bash scripts/mac/setup-local-ipfs.sh
bash scripts/mac/start-local-ipfs.sh
```

### 2.7 独立验真工具
```bash
bash scripts/mac/start-verifier.sh
```

访问地址：`http://127.0.0.1:3010`

### 2.8 macOS 一键演示环境
```bash
bash scripts/mac/start-demo.sh
```

该脚本会清理并占用 `8545`、`3005`、`3000`、`3002`、`3001`、`3010` 端口，并在多个 Terminal 窗口中启动服务。

## 3. 数据辅助脚本

| 脚本 | 用途 |
|---|---|
| `scripts/seed-local-listings.js` | 向 Local 数据库写入演示用户和北京房源数据。 |

使用方式：

```bash
node scripts/seed-local-listings.js
node scripts/seed-local-listings.js --clear
```

`--clear` 会先删除由脚本写入的 Local 演示房源，再重新写入数据。

## 4. 服务组合

| 场景 | 脚本组合 |
|---|---|
| Sepolia 演示 | `start-sepolia-services` |
| Local 演示 | `start-persistent-local-node` + `start-local-services` |
| 双环境演示 | `start-persistent-local-node` + `start-parallel-services` |
| macOS 一键演示 | `start-demo` |
| 房源图片与评论材料验真 | `start-local-ipfs` + 对应业务服务 |
| 独立验真 | `start-verifier` |

## 5. 端口汇总

| 端口 | 服务 |
|---|---|
| `3000` | Sepolia 后端 |
| `3001` | 前端 |
| `3002` | Local 后端 |
| `3005` | 认证服务 |
| `3010` | 独立验真工具 |
| `5001` | IPFS API |
| `8080` | IPFS Gateway |
| `8545` | Local EVM RPC |
